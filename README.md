Installing
==========

Add this to your composer.json:

```json
"repositories": [
    {
        "type": "vcs",
        "url": "https://github.com/FATCHIP/framework-components"
    }
],
"require": {
    "FATCHIP/framework-components": "dev-master"
}
```
