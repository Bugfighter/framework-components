Installing
==========

Add this to your composer.json:

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/FATCHIP-GmbH/fat-php-framework"
        }
    ],
    "require": {
        "FATCHIP-GmbH/fat-php-framework": "dev-master"
    }
}
```
